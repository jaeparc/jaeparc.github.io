🌀 PORTAL 2D 🌀

Portal2D is a prototype of a Portal game as a 2D platformer.

Portal2D was a project I made to answer a coding exercise around the spritesheet animation in my Unity class in first year of Game Design.

I don't have much time to work on it these days so it's kind of in standby but it will be upgraded because I think I can turn it into something enjoyable.

Credits to :
- Kenney.nl for the buildings
- Pixel_Salvage for the car sprites
- Craftpix.net for the characters sprites
and a lot of other internet artists.
