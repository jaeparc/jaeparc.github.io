🧟 FARM PROJECT 🧟

FarmProject is a small project of a Tactical-RPG taking place in a post-apo universe where you fight zombies.

It was originally just an exercise in my coding class during my second year of game design to study pathfinding on a grid but I had a lot of fun working on it so I decided to add quite a few features. I was alone on this project but I got all the assets on the internet.

Since it was a project I worked on for fun, I decided to inspire myself of stuff I thought cool of some other games even if for some, can be irrelevant.
I tried to inspire myself from the menu of Overcooked with the player roaming around a map to go from stages to stages.
For the pause menu, I tried to imitate the Tomb Raider 1 inventory menu which was really a real hassle to code.
For the end level screen, I took Broforce's one as a model.
For the aiming screen, I inspired myself from Fallout's VATS.
And even if it's not present yet, I want to do something like the animation of CoD zombie for the barriers when the player finishes a level and unlocks a new zone.

I don't have much time to work on it these days so it's kind of in standby but it will be upgraded because I think I can turn it into something enjoyable.
